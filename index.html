Neon OS 3.2
FEBRUARY 2026
Safari Compatibility Edition
Full Safari browser support — Neon OS now detects Safari and applies specific optimizations to prevent UI issues like unintended zoom on double-tap, scroll bounce overscroll, and viewport glitches.
Double-tap zoom disabled on all interactive elements. Text fields no longer trigger page zoom on iOS Safari.
Battery status sync issue resolved. Safari's limited Battery API access is now gracefully handled.
Smooth scroll and momentum scrolling optimized for iOS Safari's rubber-band scroll behavior.
Neon Keyboard
Complete Neon Keyboard overhaul — system keyboard is now fully suppressed on all inputs. Only the custom Neon Keyboard is shown inside the OS.
Keyboard push animation now properly shifts the conversation view upward without jarring layout shifts.
Keyboard haptics can now be enabled/disabled independently from global haptics.
Messages
Photo & file sharing added. Tap the attachment button in any conversation to send images, documents, audio, and more without leaving Neon OS.
Unread message badge now appears on the Messages app icon on the home screen.
Message storage optimized — older messages are automatically trimmed to keep database size minimal while preserving recent history.
Removed pulsing animation on Messages app icon (previously the .pulse class) for a cleaner home screen.
Settings
Glow Intensity now correctly applies to all outgoing message bubbles and UI glow effects in real-time.
Haptic Touch system added — every button tap triggers subtle vibration feedback (vibration API). Can be disabled globally.
Wallpaper customization added to Appearance settings with 6 unique wallpaper options.
New Settings sections: Accessibility (text size, reduce motion, bold text), Privacy (camera, storage permissions), Sounds & Haptics (combined and fully functional).
Auto-Lock now configurable: 1, 2, 5, 10, or 30 minutes.
All notification toggles now fully functional — banners, sounds, and system notifications work independently.
Calling
Call system and call buttons fully removed from the UI per OS 3.2 spec. This reduces app complexity and power usage.
Read Receipts & Polish
DM read receipts added — one grey check means sent, two grey checks means delivered, two blue checks means the recipient has read your message. Fades to blue smoothly. Groups show single check only.
All emoji icons replaced with custom Neon OS SVG symbols throughout the OS — attach menu, effect picker, update notes, and group avatars now use vector icons that match the Neon OS visual language.
Button click sounds now fire on every UI interaction including the lock pad, settings rows, and navigation buttons. AudioContext is resumed on demand to prevent silent buttons.
Bug Fixes
Fixed: Glow CSS variable not propagating to bubbles and shadows.
Fixed: System keyboard appearing over Neon Keyboard on mobile.
Fixed: Notification toggles saving but not applying correctly at runtime.
Fixed: Message badge count not updating on home screen.
